# Stock-Market-Predictive-Analytics
Rule-based stock market prediction using moving averages and RSI


Stock Market Predictive Analytics

Project Overview

This project focuses on predicting stock market movements using historical data and technical indicators. A rule-based approach is used instead of machine learning to keep the model simple and interpretable.

Objectives

* Analyze historical stock price data
* Identify trends using technical indicators
* Predict stock movement (Up/Down)
* Evaluate model performance

Tools & Technologies

* Python
* Pandas
* NumPy
* Matplotlib

Dataset

* Source: Yahoo Finance (via Kaggle)
* Stocks used: AAAU, AADR

Methodology

* Data preprocessing and cleaning
* Feature engineering (Moving Average, RSI)
* Rule-based prediction model
* Performance evaluation using accuracy

Results

* Model achieved moderate accuracy (~50%)
* Performance varied across stocks
* Moving averages helped identify trends

Limitations

* No external factors considered
* Rule-based model has limited accuracy
* Market volatility affects predictions

Future Scope

* Implement machine learning models
* Add more indicators
* Use real-time data

* Jupyter Notebook (.ipynb)
